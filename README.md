# 🔐 PassGen

Hardware-entropy password generator in C++.

## Build
### bash g++ -std=c++17 main.cpp -o passgen

## Features

- Choose character types (uppercase, lowercase, numbers, special chars)
- Custom password length
- Generate multiple passwords at once
- Password strength checker (Weak / Medium / Strong)
- Auto-copy to clipboard (Windows)
- Save all passwords to file (`password.txt`)
- Standalone `.exe` (no IDE needed to run)
